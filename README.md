# TestApp — Sample Product Codebase

This is a dummy product codebase used to test ProdPilot's 
codebase indexing and requirement analysis.

## Stack
- Backend: Python (FastAPI), Java (Spring Boot)
- Frontend: React (JSX)
- Payments: Card only (UPI not yet implemented)
- Notifications: Email + in-app (SMS not yet implemented)

## Structure
backend/auth/          User authentication
backend/payments/      Payment processing (card only)
backend/users/         User management (Java)
backend/notifications/ Email and in-app notifications
frontend/src/          React components and API client
